Personal Employment Search Assistant is an
application that keeps you and your job search
in sync. The best app for tracking job applications,

Match

Match is an Express and Node friend-finder application
meant to simulate a simple dating application. This full-stack
application matches users based on personality. It gets results on users' surveys, then compare their answers with those from other users and looks for a match.

  • HTML5, Bootstrap, Javascript, Node, Express, Path, Heroku, Git.
  • Github Repository
